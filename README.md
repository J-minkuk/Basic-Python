# 머리를 식히는 Python
* 파이썬은 **인터프리트 언어(Interpreted Language)**
* 파이썬에서는 소스 파일을 **모듈(module)** 이라고 부른다.
* 파이썬은 모든 것이 **객체(object)** 다.

## 자료형 (Data Type)
* Integer (정수)
* Floating-Point (실수)
* String (문자열)
    * 파이썬에서 한번 작성된 문자열은 변경이 불가능하다.
        > word = 'python'<br>
        word[0] = 'C' # TypeError: 'str' object does not support item assignment
